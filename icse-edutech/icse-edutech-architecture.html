ICSE Edutech Platform — India-Scale Architecture

B.L.U.E. System adapted for ICSE Class 8–10 • 5M+ Concurrent • 99.95% SLA • Multi-Region India

Scale Architecture — 5M Concurrent

  • • CloudFront: 6 India PoPs — 90% cache hit ratio on static content
  • • PWA offline: Service Worker + IndexedDB — works on 2G/3G
  • • HPA autoscaling: 150+ pods per service at peak (exam season)
  • • Read-heavy: CQRS — content reads from CDN, writes async via SQS
  • • Connection pooling: PgBouncer — 20K connections → 500 DB connections
  • • Regional sharding: Mumbai primary, Hyderabad read replicas

B.L.U.E. Core — Graph Engine

  • • Neo4j stores prerequisite DAG: Class 8 → 9 → 10 dependency chains
  • • Walkthrough Engine computes personalized paths: "What you need for Trigonometry"
  • • Graph precomputed hourly — real-time queries hit Redis cache
  • • Syllabus-mapped: Each guide tagged with ICSE chapter/section codes
  • • Gap analysis: Compare student progress against walkthrough prerequisite set
  • • Adaptive: Re-rank paths based on assessment performance

India-Optimized Infrastructure

  • • Primary: AWS ap-south-1 (Mumbai) — 3 AZs, <20ms to major cities
  • • DR: AWS ap-south-2 (Hyderabad) — warm standby, RPO <1s
  • • CDN: 6 India edge locations covering all metro + tier-2 cities
  • • Auth: Phone OTP (99% of Indian users have mobile, 60% no email)
  • • Payments: Razorpay — UPI, Paytm, PhonePe, Cards, NetBanking
  • • Cost optimization: Spot instances for non-critical workloads, reserved for baseline

Data Strategy — PostgreSQL

  • • Users: Phone + OTP auth, class/board, progress state
  • • Content: Guide metadata, syllabus mapping, version history
  • • Progress: Completed guides, assessment scores, streak data
  • • Subscriptions: Stripe-style recurring billing via Razorpay
  • • Read replicas: 3x read replicas for dashboard queries
  • • Partitioning: By class (8/9/10) + board (ICSE) for query perf

Security — Defense in Depth

  • • WAF: OWASP Top 10 + rate limiting (100 req/s per IP)
  • • Shield Advanced: DDoS protection at edge
  • • Auth: OTP → JWT (15min access + 7d refresh), phone-verified
  • • Network: All services in private subnets, ALB-only ingress
  • • Data: Encryption at rest (KMS) + transit (TLS 1.3)
  • • Content: Verifier-reviewed before publish (B.L.U.E. model)

Monetization Architecture

  • • Freemium: Free walkthroughs up to Class 8, paywall at 9–10 exam prep
  • • Individual: ₹299/mo or ₹1,999/yr — full access + mock tests
  • • School licensing: ₹50K–₹2L/yr per school — bulk seats + admin dashboard
  • • Coach upsell: Walkthrough identifies gap → "Book 1:1 ₹199 session"
  • • Razorpay Subscriptions: Automated billing, dunning, UPI auto-debit
