# Tech Stack Reference

_Last updated: 2026-06-18_

## Frontend (apps/web)
- Next.js 15.2.6
- React 19
- TypeScript
- Tailwind CSS 3.4
- lucide-react (icons)
- clsx (classnames)

## Backend (apps/api)
- Node.js + TypeScript
- Express 4
- JWT (jsonwebtoken)
- zod (validation)
- cors, helmet, morgan
- ts-node-dev (dev server)

## Graph Engine (packages/graph-engine)
- Go
- Custom graph library at `graph/`
- HTTP server (net/http)

## Tooling
- TypeScript across JS packages
- npm workspaces (monorepo)
- go.mod for Go module
